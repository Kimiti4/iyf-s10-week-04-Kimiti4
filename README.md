## Project Description
This project contains my Week 4 JavaScript practice work. It covers JavaScript basics such as variables, data types, operators, functions, control flow, arrays, objects, and arrays of objects.

It also includes two mini-projects:
- Interactive Calculator
- Student Grade Tracker

## Files
- `index.html` - main HTML file
- `main.js` - JavaScript practice and mini-projects

## Concepts Covered
- Variables (`let`, `const`)
- Data types
- Operators
- Functions
- If/else and switch
- Loops
- Arrays and array methods
- Objects
- Arrays of objects

## Mini Projects
### 1. Calculator
A console-based calculator supporting:
- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Power

### 2. Student Grade Tracker
A grade management system that can:
- Add students
- Calculate student averages
- Calculate subject averages
- Find the top student
- Identify struggling students
- Generate report cards
